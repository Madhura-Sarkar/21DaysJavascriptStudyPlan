# 🎯 Number Guessing Game

A simple and interactive **Number Guessing Game** built using **HTML, CSS, and JavaScript**.
This project is part of my JavaScript learning journey, where I focus on building small, creative, and practical applications.

---

## 🚀 Features

* 🎮 Interactive UI (no prompt/alert)
* 🔢 Random number generation (1–100)
* 📉 “Too High” / 📈 “Too Low” hints
* 🧮 Attempts counter
* 🔄 Restart game functionality
* ❌ Input validation

## ▶️ How to Run

1. Download or clone the repository
2. Open the `index.html` file in your browser
3. Start playing 🎉

---

## 🎯 How to Play

1. Enter a number between **1 and 100**
2. Click **Submit**
3. Get hints:

   * 📉 Too High
   * 📈 Too Low
4. Keep guessing until you win
5. Check how many attempts you took!

---

## 🔄 Restart Game

Click the **Restart** button to:

* Reset attempts
* Generate a new random number
* Clear previous results

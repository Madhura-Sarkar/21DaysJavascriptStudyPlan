# 🔤 Word Scramble Game

A simple and interactive Word Scramble Game built using **HTML, CSS, and JavaScript**.  
The game displays a scrambled word, and the user has to guess the correct word.

---

## 🚀 Features

- 🎲 Random word selection
- 🔀 Word scrambling logic
- ⌨️ User input to guess the word
- ✅ Instant feedback (Correct)
- 📊 Score tracking
- 🔁 New word generation

---

## 🧠 Concepts Used

This project helps practice:

- **Strings** → `split()`, `join()`
- **Arrays** → storing words, shuffling
- **Math.random()** → random word selection
- **DOM Manipulation** → updating UI dynamically
- **Functions & Logic Building**

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## ⚙️ How It Works

1. A random word is selected from an array
2. The word is scrambled using:
   - `split("")` → convert to array
   - `sort()` → shuffle
   - `join("")` → convert back to string
3. User enters their guess
4. Input is compared with the original word
5. Score updates if correct

---

## ▶️ How to Run

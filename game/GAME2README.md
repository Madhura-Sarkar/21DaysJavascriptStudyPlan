# 🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.
This project is part of my JavaScript learning journey to practice DOM manipulation, events, and game logic.

---

## 🚀 Features

* 🎯 Play Rock, Paper, Scissors against the computer
* 🤖 Random computer choice logic
* 📊 Match tracking system:

  * Total Matches Played
  * Wins
  * Losses
* 🔁 "Play Again" button to reset the game
* 🎨 Clean and centered UI design

---

## 🧠 What I Learned

* JavaScript **functions & parameters**
* **Event handling** using `onclick`
* **Random number generation** (`Math.random`)
* **Conditional logic** (`if/else`)
* **DOM manipulation** (`getElementById`)
* Basic **CSS layout (Flexbox)**

---

## 🕹️ How to Play

1. Click on **Rock**, **Paper**, or **Scissors**
2. The computer will randomly choose its move
3. The result will be displayed:

   * ✅ Win
   * ❌ Lose
   * 🤝 Draw
4. Scores update automatically
5. Click **Play Again** to reset the game

# 🔢 Simple Calculator (Game 3)

A basic calculator built using **HTML, CSS, and JavaScript**.
This project performs simple arithmetic operations and helps practice core JavaScript concepts.

---

## 🚀 Features

* Add, Subtract, Multiply, Divide
* User-friendly input fields
* Dropdown to select operation
* Displays result instantly
* Handles invalid input
* Prevents division by zero
* Centered UI layout

---

## 🧠 Concepts Used

* Functions
* `switch` statements
* DOM Manipulation (`getElementById`)
* Event handling (`onclick`)
* Input validation (`isNaN`)
* Basic CSS (Flexbox for centering)

---

## 🖥️ How It Works

1. User enters two numbers
2. Selects an operation (+, -, *, /)
3. Clicks **Calculate**
4. Result is displayed on the screen

---

## ⚠️ Error Handling

* Shows message if inputs are empty or invalid
* Prevents division by zero

---

## 📂 Project Structure

